# Install MindGw-w64

### 1. Download Packages
Download MSYS2 from https://www.msys2.org/

### 2. Install MSYS2
Install MSYS2 and follow the installation instructions

### 3. Open MSYS2 terminal
Open the MSYS2 terminal and run :

`pacman -S mingw-w64-x86_64-gcc`
`pacman -S mingw-w64-x86_64-cmake`
`pacman -S mingw-w64-x86_64-make`

### 4. Add to PATH :

Add to your Windows PATH : `C:\msys64\mingw64\bin`

### 5. Restart your VsCode
